Cómo se mantiene ahora:
1. Para cambiar el menú, logo superior, barra superior o navegación desktop, edita:
   components/header.html

2. Para cambiar el footer, contacto, links legales o empresa, edita:
   components/footer.html

3. Para cambiar la navegación inferior móvil, edita:
   components/bottomnav.html

4. No edites header/footer dentro de cada HTML. Cada HTML solo contiene:
   <div data-component="header"></div>
   <div data-component="footer"></div>
   <div data-component="bottomnav"></div>
